# SilkEcon

SilkEcon is an AI-powered financial planning platform designed specifically for Uzbekistan.

## Features

- Real Life Budget Calculator
- Loan Trap Detector
- Can I Afford It Simulator
- AI Financial Advisor

## Mission

To help people in Uzbekistan make better financial decisions using accessible digital tools and AI.

## Technologies

- HTML
- CSS
- JavaScript
- Anthropic API
